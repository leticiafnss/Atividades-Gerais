📚 Exercícios de JavaScript

Este repositório contém exercícios práticos desenvolvidos durante meus estudos de JavaScript, com foco em lógica de programação e estruturas fundamentais da linguagem.

🚀 Conteúdos praticados

Entrada e saída de dados

Operadores lógicos e aritméticos

Estruturas condicionais (if / else)

Laços de repetição

for

while

do...while

Vetores (arrays)

🧠 Exercícios implementados
Lista 01 — Laço FOR

Programa que lê 10 números inteiros digitados pelo usuário e mostra:

Quantos números são pares

Quantos números são ímpares

Lista 02 — Laço WHILE

Programa que lê idades de várias pessoas até que uma idade negativa seja digitada e mostra:

Quantas pessoas têm menos de 21 anos

Quantas pessoas têm mais de 50 anos

Lista 03 — Laço DO...WHILE

Programa que lê números inteiros até que o número 0 seja digitado e calcula:

A soma de todos os números positivos

Lista 04 — Vetores

Programa que possui um vetor com 10 números inteiros e permite:

Buscar um número digitado pelo usuário

Mostrar a posição do número no vetor

Informar quando o número não for encontrado

🛠 Tecnologias utilizadas

JavaScript

Node.js

readline-sync

📖 Objetivo

Este repositório faz parte da minha jornada de aprendizado em programação e desenvolvimento de software, com foco em fortalecer minha lógica de programação e prática com JavaScript.
